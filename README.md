# OAU-WGTBAM
 Obafemi Awolowo University, Who Gets To Be A Millionare

---

## Routes

| Route | Who | Purpose |
|---|---|---|
| `/` | Player | Waiting screen — sits here until admin starts the game |
| `/challenge` | Player | Live question display |
| `/checkpoint` | Player | Withdraw or continue decision |
| `/total` | Player | Final prize amount with confetti |
| `/prize` | Player | Prize ladder overlay |
| `/registration` | Public | Event registration form |
| `/admin` | Admin | Start the game |
| `/admin/challenge` | Admin | Controls — click answers, reveal correct, next question |
| `/admin/checkpoint` | Admin | Controls — withdraw or continue |
| `/admin/total` | Admin | Final screen + Reset Quiz button |
| `/admin/prize` | Admin | Prize ladder overlay |
| `/admin/registration` | Admin | Registration form UI (placeholder) |
| `/analytics/users` | Admin | Table of all registered users |
